# Personal Website

This is a simple personal website built with native HTML, CSS, and a tiny amount of JavaScript.

## Local preview

Open `index.html` directly in your browser for a quick preview.

If you want a local server, run one of these commands in this folder:

```powershell
python -m http.server 8000
```

or

```powershell
py -m http.server 8000
```

Then open:

`http://localhost:8000`

## Before publishing

- Add your CV PDF at `cv/CV_CCLOOS.pdf`.
- Replace the example project pages with your final project content when ready.

## GitHub Pages later

Push this repository to GitHub, then enable GitHub Pages from the repository settings and publish from the `main` branch.
